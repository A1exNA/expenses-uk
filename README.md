# Учёт расходов управляющей компании (УК)

Веб-приложение для учёта расходов на текущий ремонт многоквартирных домов. 
Разрабатывается на стеке: React (фронтенд), PHP (бэкенд), MySQL.

## Структура базы данных
- objects – дома
- users – сотрудники
- spending_groups – группы расходов
- bills – счета (безналичные)
- expense_bills – позиции счетов
- checks – чеки (наличные)
- expense_checks – позиции чеков
- deposits – выдача подотчётных средств

## Установка
1. Склонировать репозиторий в папку локального сервера (например, OpenServer, MAMP, XAMPP).
2. Создать базу данных `expenses_uk` и импортировать `database/schema.sql` (будет добавлен позже).
3. Настроить подключение в `backend/config/database.php`.
4. Настроить веб-сервер так, чтобы корень указывал на папку `backend/` (или настроить .htaccess).
5. Открыть в браузере `http://expenses-uk/backend/`.
